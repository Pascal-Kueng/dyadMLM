# Summarize prepared dyadic data

Prints the dyadic structure followed by standard summaries for all
columns.

## Usage

``` r
# S3 method for class 'dyadMLM_data'
summary(object, ...)
```

## Arguments

- object:

  A `dyadMLM_data` object returned by
  [`prepare_dyad_data()`](https://pascal-kueng.github.io/dyadMLM/reference/prepare_dyad_data.md).

- ...:

  Arguments passed to
  [`summary()`](https://rdrr.io/r/base/summary.html).

## Value

Invisibly, the standard summary of all columns.
